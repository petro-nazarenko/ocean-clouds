Power users: Claude Code plugin

Install the ocean-clouds plugin for Claude Code and search directly from your terminal.

1

Clone & install

git clone https://github.com/petro-nazarenko/ocean-clouds
cd ocean-clouds
pip install -r requirements.txt
playwright install chromium
2

Configure Telegram (optional)

cp .env.example .env
# Add TG_API_ID and TG_API_HASH
python -m scraper.telegram --setup
3

Use in Claude Code

/ocean-clouds:search Chief Officer Tanker
/ocean-clouds:tg --since 48
/ocean-clouds:filter --salary 7000 --urgent
/ocean-clouds:sources
